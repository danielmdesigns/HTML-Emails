Simple Responsive Table
What makes tables responsive with the the least amount of markup & what elements are neccessary to use when coding HTML Emails for the future?

1.) first, you must include the meta viewport & content width=device-width in the head.

2.) set body to 100% width

3.) then, the main table container also needs to be 100% width.

4.) ALL individual td cells, are max-width of 600px, but you have to use the !important declaration and also use the display:block; property.

5.) img tags need a max-width of 100% and height of auto, placed inside of the td cell

6.) for body copy, put text into a div w/ right & left padding to give it space

7.) finally, for 2cols -- you need to nest a table w/ media queries inside a tr

that's it! that's all we should need in order to have responsive tables!
